## Unit 3 - Adjectives and conjunctions
&nbsp;
- tantus, tanta, tantum - of such size, so much, so great
- talis, tale - so great
- tot - so many
- quamquam (+ ind.) - although
- quamvis (+ subj.) - although
- licet (+ subj.) - althoguh
- quod - because, since
- quia - because, since
- cum (+ subj.) - when, since, although
